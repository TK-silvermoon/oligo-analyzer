Tm 計算方法
① Wallace則

短鎖オリゴ（<20 mer）向けの簡易式。GC含量のみを考慮。

Tm (°C) = 2(NA + NT) + 4(NG + NC)

N = 各塩基の数。精度は ±5°C 程度。高塩・短鎖での目安に使用。

Suggs SV et al. (1981) ICN-UCLA Symp Mol Cell Biol 23:683–693
② 最近傍法（Nearest-Neighbor法）

隣接塩基対の積み重なり相互作用（スタッキング）を考慮した高精度法。

Tm (°C) = ΔH × 1000 / (ΔS + R × ln(CT/4)) − 273.15

ΔH：最近傍エンタルピー（kcal/mol）の総和＋末端補正
ΔS：最近傍エントロピー（cal/mol/K）の総和＋末端補正
R：気体定数 = 1.987 cal/mol/K
CT：オリゴ濃度（mol/L）

SantaLucia J Jr. (1998) Proc Natl Acad Sci USA 95(4):1460–1465

塩濃度補正（Owczarzy法）

Tm([Na⁺]) = Tm(1M NaCl) + 16.6 × log₁₀([Na⁺])
Owczarzy R et al. (2004) Biochemistry 43(12):3537–3554

消光係数・濃度計算
ε₂₆₀ — 消光係数

1本鎖DNAの260 nmにおける消光係数を塩基組成から計算。

ε₂₆₀ = nA×15400 + nT×8700 + nG×11500 + nC×7400

単位：L / (mol · cm)。ランダム配列の長鎖では±10%程度の誤差。

Warshaw MM & Tinoco I Jr. (1965) J Mol Biol 13(1):54–64
Beer–Lambert則による濃度換算
A₂₆₀ = ε × c × l c (µM) = A₂₆₀ × 10⁶ / ε c (ng/µL) = c (µM) × MW / 1000

l：光路長 = 1 cm（NanoDropは自動換算）
MW：分子量（g/mol）
nmol = c (µM) × volume (mL)

二次構造解析
ヘアピン ΔG 推定
ΔG(37°C) = ΔG_stem + ΔG_loop

ΔG_stem：ステム部の最近傍法（SantaLucia 1998）
ΔG_loop：ループ長に応じたペナルティ（Zuker近似）

判定基準：
ΔG > 0 kcal/mol → 低リスク（緑）
−2 < ΔG ≤ 0 → 要注意（オレンジ）
ΔG ≤ −2 → 高リスク（赤）

ダイマー 連続相補塩基数の目安

PCRへの影響は 3'末端の関与が最も重要。

連続相補 ≥ 3 bp → 要注意
連続相補 ≥ 5 bp → 高リスク
3'末端関与 ≥ 3 bp → PCR阻害の可能性大

Vallone PM & Butler JM (2004) BioTechniques 37(2):226–231
